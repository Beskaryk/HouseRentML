# House Price Prediction Mini-Project

Мини-проект по задаче **прогнозирования стоимости домов** на основе табличных признаков.

## Описание проекта

Цель проекта — построить и сравнить несколько моделей машинного обучения для задачи **регрессии**: предсказать стоимость жилого объекта по его характеристикам.

В рамках проекта были рассмотрены как базовые линейные модели, так и регуляризованные и ансамблевые методы. Особое внимание уделено корректной предобработке данных, логарифмической трансформации целевой переменной, подбору гиперпараметров и сравнению качества моделей по метрикам **RMSE** и **MAE**.

## Используемый датасет

В работе используется датасет **House Prices / Ames Housing Dataset**.

Целевая переменная:
- `SalePrice` — цена продажи дома.

Признаки включают:
- площадь объекта;
- качество и состояние дома;
- количество комнат;
- наличие гаража, подвала и других элементов;
- характеристики участка;
- район и другие параметры недвижимости.

## Задачи проекта

В ходе работы были выполнены следующие этапы:

1. Загрузка и предварительная подготовка данных;
2. Выделение признаков и целевой переменной;
3. Логарифмическая трансформация `SalePrice`;
4. Обработка пропусков в числовых и категориальных признаках;
5. Масштабирование числовых признаков и one-hot кодирование категориальных;
6. Разбиение выборки на обучающую и отложенную части;
7. Обучение базовых моделей;
8. Подбор гиперпараметров с помощью `GridSearchCV`;
9. Сравнение моделей по метрикам `CV RMSE (log)`, `Holdout MAE` и `Holdout RMSE`;
10. Анализ результатов и выбор лучшей модели.

## Использованные модели

В проекте были рассмотрены следующие алгоритмы:

- **Linear Regression**
- **Ridge Regression**
- **Lasso Regression**
- **Random Forest Regressor**

## Лучшая модель

По результатам сравнения моделей наилучшее качество показала модель **Lasso Regression с подбором гиперпараметров**.

Она получила минимальное значение `Holdout RMSE` (`21105.181866`) и минимальное значение `Holdout MAE` (`14595.224572`) среди всех рассмотренных моделей. Это говорит о том, что после подбора гиперпараметров Lasso оказалась наиболее эффективной для задачи прогнозирования стоимости домов на данном датасете.

Модель `Ridge (params)` также показала очень сильный результат и оказалась близка к лучшей по всем основным метрикам. Базовая `Linear Regression` выступила как хороший baseline, но уступила регуляризованным моделям. `Random Forest Regressor` в рамках данного проекта показал более высокую ошибку на отложенной выборке, а `Lasso` без подбора параметров продемонстрировала худший результат из-за слишком сильной регуляризации по умолчанию.
